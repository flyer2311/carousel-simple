# carousel-simple
for self
调用函数为carousel(id)
如果要为轮播改变颜色样式,可以添加如下class样式
carousel-primaryBlue(蓝色)
carousel-infoCyan(青色)
carousel-successGreen(绿色)
carousel-warnOrange(橙色)
carousel-dangerRed(红色)
(默认黑白色)
轮播支持功能：左右点击，
                      子按钮点击，
                      鼠标滑入暂停，
                      鼠标滑出继续
注意:必须设置的值，id对应得class的width和height
